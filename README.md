# ノベルゲーム

## 概要 
このサイトは僕たちが製作したノベルゲームを展示するためのサイトです。<br>
ノベルゲームの軽い説明と実際のゲームのサイトに飛ぶとこができるようになっています。<br>
製作した作品の写真をクリックしてもらうと、実際にゲームを遊ぶことができます。

## ゲームの説明
  #### ノベルゲームとは、<br>
  自分が物語の「主人公」になって、たくさん選択肢から自分だけの選択で物語を見届けるゲームです。<br>
  小説みたいなものですがイラストやアニメーション、BGMなど<br>
  小説のゲーム版みたいな感じなので本が苦手な方でも楽しめます!!!<br>
  本が好きな方は本とは違う感覚なのではまってしまうかもしれません！<br>
 ## ゲームの遊び方
  ゲームの遊び方は物語を読んで、途中に出てくる選択肢をクリックするだけです。<br>
  基本的に画面をクリックするだけで遊べます。<br>
  簡単に遊べるところが魅力です。<br>
